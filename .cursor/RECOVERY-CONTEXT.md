# Master Tracker Quick Recovery (Non-Technical)

If Cursor is reinstalled or old chat is lost, open this project folder and paste:

`Use skill master-tracker-recovery and restore my master tracker setup.`

## Project location

- `Desktop/zebronics/zebronics-master-tracker`

## What this app does

- Uploads daily Excel files for Zebronics monitor/projector data.
- Shows Amazon/Flipkart dashboards and PO guidance.
- Uses Supabase backend.

## Before prompting

1. Open this folder in Cursor.
2. Confirm `.env.local` exists.
3. If app does not start, run `npm install`.

## Normal commands

- Start app: `npm run dev`
- Check changes: `git status`
- Pull latest: `git pull --rebase origin main`

## Important safety

- Never share tokens/keys in chat.
- Never commit `.env.local`.
- Never force push unless you explicitly ask for it.
